lagotto-rb

This is alpha software, so expect changes

lagotto-rb - a Ruby client for the Lagotto application for article level metrics data

Dependencies

  • HTTParty gem to make web calls to Crossref APIs
  • json gem to convert to/from JSON
  • bundler
  • rake

Install

Release version

gem install lagotto-rb

Development version

Install dependencies

gem install httparty json rake
sudo gem install bundler
git clone https://github.com/lagotto/lagotto-rb.git
cd lagotto-rb
bundle install
